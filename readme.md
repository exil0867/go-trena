# go-trena  

**go-trena** is an open-source training app for tracking and optimizing workouts, nutrition, and progress.  

## Features (Planned)  
- Cardio tracking  
- Resistance training tracking  
- Nutrition logging  
- Weight tracking & progress monitoring  
- Stats & insights  

## Tech Stack  
- **Backend:** Go (Fiber, Supabase)  
- **Frontend:** Flutter
